# Personal Web Portfolio - Nadhif Rif'at Rasendriya

Website portofolio pribadi sederhana namun profesional untuk menampilkan profil, keahlian, proyek, dan kontak. Proyek ini dibangun sebagai contoh hasil kerja Web Developer Intern di Socialvit (PT Talenta Sinergi Group).

## 🎯 Tujuan Proyek
Membuat website portofolio interaktif yang menonjolkan kemampuan pengembangan antarmuka pengguna, sekaligus mempraktikkan dasar HTML semantik dan struktur layout yang rapi.

## 🛠️ Teknologi yang Digunakan
* **HTML5:** Struktur semantik web.
* **Tailwind CSS (CDN):** Styling dan utilitas desain responsif (tanpa framework CSS eksternal tambahan).
* **Vanilla JavaScript:** Logika interaktivitas dan manipulasi DOM (untuk fitur selanjutnya).
* **Google Fonts:** Tipografi (Inter).

## 🚀 Cara Menjalankan
Karena proyek ini menggunakan HTML statis dan Tailwind CSS via CDN, tidak diperlukan instalasi package khusus (seperti Node.js/NPM).
1. Clone repositori ini: `git clone https://github.com/nadhif-royal/nadhif-portfolio.git`
2. Buka folder proyek di Text Editor (misal: VS Code).
3. Buka file `index.html` langsung di browser, atau gunakan ekstensi **Live Server** di VS Code untuk pengalaman yang lebih baik.
